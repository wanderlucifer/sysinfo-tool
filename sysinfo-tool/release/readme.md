## 📥 下载与安装

### 1. 下载程序

根据您的操作系统选择对应的版本：

| 系统架构 | 文件名 | 适用环境 |
| :--- | :--- | :--- |
| **64 位系统** | `sysinfo-tool_x64.exe` | Windows 10/11 (64-bit) |
| **32 位系统** | `sysinfo-tool_x86.exe` | Windows 10/11 (32-bit) |

> 💡 **小贴士**：如果不确定自己的系统位数，可以**两个都试试**，通常能成功运行一个。

### 2. 运行程序

双击下载后的 `.exe` 文件即可启动工具。

## ⚠️ 常见问题 & 注意事项

### 🦠 为什么杀毒软件会报警？

**这是误报！请放心。**

在使用本工具时，如果您的杀毒软件（如 360、火绒、Windows Defender 等）弹出“威胁警告”或拦截提示：

*   **原因**：软件尚未获得微软官方数字签名认证（Open Source tools often face this）。
*   **解决办法**：请**添加白名单**或暂时关闭杀毒软件后再次运行。

### ❓ 程序闪退或无法启动？

1.  **检查架构**：确认是否选错了系统版本（尝试 32 位换 64 位）。
2.  **运行环境**：
    请确保安装了 VC++ Redistributable。
3.  **权限问题**：尝试右键点击程序，选择 **“以管理员身份运行”**。
