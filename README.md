# 🛠️ SysInfo Tool - 系统信息采集工具

<div align="center">

一个简单、快速、好用的系统信息查询工具，让您的电脑状态一目了然。
<br>
**Apache License 2.0** &bull; **Windows x64/x86** &bull; **兼容Windows XP ~ Windows 11**

</div>

## ✨ 特性

- 🚀 **即开即用**：无需安装，解压双击即可运行。
- 🖥️ **双架构支持**：提供 32 位 (x86) 和 64 位 (x64) 版本，覆盖主流系统。
- 🏛️ **XP友好**：32位版本兼容 Windows XP 及以上系统。
- 🛡️ **安全透明**：开源代码，Apache 2.0 协议保护。
- 🖼️ **本地化体验**：中文界面，直观显示系统关键信息。
- 📊 **Excel导出**：一键导出系统信息到Excel表格。

### 📋 采集信息项

| 信息项 | 说明 |
| :--- | :--- |
| 责任民警 / 所属单位 | 用户可填写 |
| CPU型号 | 读取注册表 CPU 型号信息 |
| 出厂日期 | 支持多种日期格式自动解析（含中文、点号、斜杠、横线、纯数字等） |
| 操作系统版本 | 含版本号、构建号、位数 |
| 系统安装时间 | 读取注册表安装日期 |
| 浏览器版本 | 自动检测 Chrome/Edge/Firefox/IE |
| IP地址 | 读取注册表网络配置 |
| MAC地址 | 读取物理网卡 MAC |
| 硬盘序列号 | 读取注册表磁盘信息 |

## 📥 下载与安装

### 1. 下载程序

根据您的操作系统选择对应的版本：

| 系统架构 | 文件名 | 适用环境 |
| :--- | :--- | :--- |
| **64 位系统** | `sysinfo-tool_x64.exe` | Windows 10/11 (64-bit) |

### 2. 运行程序

双击下载后的 `.exe` 文件即可启动工具。

## ⚠️ 常见问题 & 注意事项

### 🦠 为什么杀毒软件会报警？

**这是误报！请放心。**

在使用本工具时，如果您的杀毒软件（如 360、火绒、Windows Defender 等）弹出"威胁警告"或拦截提示：

*   **原因**：软件尚未获得微软官方数字签名认证。
*   **解决办法**：请**添加白名单**或暂时关闭杀毒软件后再次运行。

### ❓ 程序双击无反应？

1.  **检查架构**：确认是否选错了系统版本（尝试 32 位换 64 位）。
2.  **后台进程**：程序有互斥锁防止重复运行，请检查任务管理器是否已有实例在运行。
3.  **权限问题**：尝试右键点击程序，选择 **"以管理员身份运行"**。
